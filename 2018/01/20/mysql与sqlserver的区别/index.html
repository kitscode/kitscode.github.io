mysql与sqlserver的区别

Jan 20, 2018

本篇记录下工作中遇到的和自己学习到的mysql与sqlserver的区别。

1.分页：mysql和sqlserver分别用limit和top关键字实现分页查询
例：取出数据库中前一百条数据
mysql: SELECT * FROM table LIMIT 100
sqlserver: SELECT TOP * 100 FROM table
但是如果查询要加上偏移mysql可能更方便点，例：每页100条，取出第三页
mysql: SELECT * FROM table LIMIT 200,100
sqlserver中只能建立子查询实现:
SELECT TOP 100 * FROM table WHERE id NOT IN (SELECTTOP 200 id FROM table ORDER BY id DESC)

2.mysql中用``区分字段名与关键字，sqlserver中用[]区分

3.sqlserver的简单查询速度不如mysql，大数据量时，mysql中最好不要使用函数/join/group等方式查询，性能降低比较多。sqlserver的复杂查询性能降低的却不多，可能sqlserver的查询优化做得更好。

4.mysql创建临时表的关键字：CREATE TEMPORARY TABLE temp(…);
sqlserver创建临时表的关键字：CREATE TABLE #temp(…);
